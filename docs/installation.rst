Installation
============

Requirements
------------

Python 3.10 to 3.14 supported.

Django 5.2 to 6.1 supported.

Installation
------------

1. Install with **pip**:

   .. code-block:: sh

       python -m pip install django-orjson

2. Add to your ``INSTALLED_APPS``:

   .. code-block:: python

       INSTALLED_APPS = [
           ...,
           "django_orjson",
       ]

Then, start adopting the features as covered in the next documentation pages!
